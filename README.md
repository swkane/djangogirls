## Get Started

1. Create your virtual environment
   `python3 -m venv [myvenv]`

2. Start your virtual environment
   `source [myvenv]/bin/activate`

3. Install dependencies
   `pip install -r requirements.txt`

4. Run Server
   `python manage.py runserver`
